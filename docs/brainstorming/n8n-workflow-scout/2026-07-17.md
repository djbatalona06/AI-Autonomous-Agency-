## Sales (SAL)

1. **Cold-Reply Classifier & Router**
   Trigger → Node chain: `IMAP/Gmail Trigger (reply inbox)` → `Claude/GPT (classify: interested / not-interested / objection / wrong-person)` → `Switch (4 branches)` → `CRM task create (interested)` / `CRM log + suppress (not interested)` → `Apollo/Clearbit re-search (wrong person)` → `Slack alert (interested, <2hr SLA)`
   Why now: pulled straight out of the "Reply Handler" pattern in [BuiltByJoey's lead-gen template pack](https://builtbyjoey.com/blog/n8n-workflow-templates-lead-generation/) — solves the #1 complaint reps have (good replies sit unread in a shared inbox).
   Fit: Medium, $2,200–$3,200.

2. **LinkedIn Job-Signal Outreach**
   Trigger → Node chain: `Schedule Trigger` → `Apify (LinkedIn job-change / hiring-signal actor)` → `Gemini/Claude (personalize opener from signal)` → `Dedupe vs CRM` → `Google Sheets / CRM queue` → `Email/LinkedIn send`
   Why now: cited in [ConnectSafely's 2026 template roundup](https://connectsafely.ai/articles/n8n-templates-workflow-automation-examples) as a rising outbound pattern — job-change and hiring signals convert far better than cold lists.
   Fit: Medium, $2,500–$3,500.

3. **CRM Database Reactivation Engine**
   Trigger → Node chain: `Schedule Trigger (monthly)` → `CRM query (no activity 90+ days)` → `Segment (customer / lead / visitor)` → `Claude (per-segment personalized reactivation copy)` → `Saleshandy/SendGrid (3-step sequence)` → `Open/reply tracker → escalate to Reply Handler above`
   Why now: same "Database Reactivation" pattern BuiltByJoey ships as a premium template — a clean upsell for existing Rung 2/3 clients sitting on a stale CRM.
   Fit: Complex, $3,500–$4,500 (pairs naturally with idea #1).

4. **Post-Call AI Summary → CRM Timeline**
   Trigger → Node chain: `Webhook (GHL/dialer call-ended)` → `Download recording` → `Whisper transcribe` → `GPT summarize + extract action items` → `CRM note create` → `Associate note with contact`
   Why now: this exact chain shows up in two independent sources this week — [NextAutomation's real-estate pack](https://nextautomation.us/resources/free-templates/6-real-estate-n8n-templates-20260209) (GHL Auto-Call Summary Writer) and general sales-ops recaps. Reusable across SAL and WHL.
   Fit: Simple–Medium, $1,800–$2,600.

5. **AI Lead Scorer & Auto-Router**
   Trigger → Node chain: `Webhook (form/CRM event)` → `HTTP Request (Apollo/Clearbit enrich)` → `Claude (score 0–100 vs. ICP criteria)` → `Switch (hot/mid/low)` → `Hot: Slack + calendar-link email` / `Mid: 24h nurture sequence` / `Low: newsletter opt-in + sheet log`
   Why now: called out repeatedly as "the SaaS-team default" in [Goodspeed Studio's agency-tested list](https://goodspeed.studio/blog/n8n-templates) — a natural upsell layered on top of the existing SAL-01 Lead Capture template.
   Fit: Medium, $2,200–$3,000.

## E-commerce (ECM)

1. **LTV-Segmented Cart Recovery**
   Trigger → Node chain: `Shopify Webhook (checkout created)` → `HTTP Request (fetch customer LTV)` → `Switch (high/mid/low LTV)` → `High: premium email + SMS + Slack alert to sales` / `Mid: email + SMS` / `Low: single recovery email`
   Why now: a direct upgrade path for the existing "spec" Abandoned Cart Recovery card — the pattern is trending in [n8nlab's Shopify roundup](https://n8nlab.io/blog/n8n-workflow-automation-shopify) and a recent [YouTube walkthrough using Supabase as the store](https://www.youtube.com/watch?v=y03IthePB1E) as the routing backbone.
   Fit: Complex, $3,000–$4,000 (positions above ECM-01).

2. **AI Cross-Sell Recommender in Recovery Flow**
   Trigger → Node chain: `Shopify Webhook (cart abandoned)` → `Shopify GraphQL (purchase history + LTV)` → `Claude/OpenAI (2–3 complementary cross-sell picks)` → `HTML template compile` → `Klaviyo/ActiveCampaign send` → `Wait 24h → Twilio SMS fallback`
   Why now: n8nlab frames this as the "enterprise-grade" version of cart recovery — AOV lift instead of just recovery rate, a strong Complex-tier pitch for $5K+/mo stores.
   Fit: Complex, $3,800–$4,800.

3. **Proactive Shipping-Exception / WISMO Reducer**
   Trigger → Node chain: `Schedule Trigger (poll carrier API)` → `HTTP Request (tracking status)` → `IF (delay/exception detected)` → `Claude (draft plain-language delay notice)` → `Email/SMS to customer` → `Google Sheets log (ticket-deflection tracking)`
   Why now: called a top-4 workflow pattern in current real-estate/e-com automation recaps for "drastic reduction in WISMO tickets" — cuts support volume before the customer even opens a ticket.
   Fit: Medium, $2,000–$3,000.

4. **Post-Recovery Retargeting Sync**
   Trigger → Node chain: `Schedule/Wait (cart still unrecovered after final touch)` → `Code (hash customer identifiers)` → `Facebook Graph API (add to Custom Audience)` → `Google Sheets log (attribution)`
   Why now: n8n's own [multi-channel cart-recovery template](https://n8n.io/workflows/11805-recover-shopify-abandoned-carts-with-email-sms-whatsapp-and-facebook-retargeting/) closes the loop this way — a clean $500–$800 add-on to any cart-recovery build rather than a standalone sell.
   Fit: Simple add-on, $600–$900.

5. **Category-Aware Reorder Win-Back**
   Trigger → Node chain: `Schedule Trigger (daily)` → `Shopify Order query` → `Code (compute per-category reorder window, not a flat 60/90-day default)` → `Filter (past window)` → `Claude (personalized offer referencing exact prior purchase)` → `Klaviyo send`
   Why now: a sharper version of the existing "spec" AI Win-Back card — flat reorder windows miscall consumables vs. durable goods, and category-aware timing is the fix multiple 2026 guides now recommend.
   Fit: Medium, $2,200–$3,200 (upgrade path for ECM-05).

   Why now: featured as the flagship investor workflow in [NextAutomation's 6-template real-estate pack](https://nextautomation.us/resources/free-templates/6-real-estate-n8n-templates-20260209) — a discovery-stage complement to the existing WHL-04 County/Motivated-List Ingestion card.
   Fit: Complex, $3,000–$4,000.

2. **High-Equity Absentee-Owner Alert**
   Trigger → Node chain: `Schedule Trigger` → `HTTP Request (BatchData API search)` → `Code (diff vs. previous scan)` → `Filter (high equity + absentee-owner flags)` → `HTTP Request (owner detail lookup)` → `Email + Slack/Teams alert with Google Maps link`
   Why now: this exact chain is n8n's own [BatchData property-lead template](https://n8n.io/workflows/3665-automated-property-lead-generation-with-batchdata-and-crm-integration/) — directly on-thesis for wholesalers hunting off-market equity spreads.
   Fit: Medium, $2,000–$3,000.

3. **AI Cash-Buyer Scoring & LLC Enrichment**
   Trigger → Node chain: `Schedule Trigger (6 AM)` → `Google Sheets (buyers_raw)` → `Claude (classify investor probability + entity type, strict JSON)` → `HTTP Request (LLC enrichment lookup)` → `Code (phone hygiene: mobile vs. landline vs. VoIP)` → `Code (1–5 buyer score: recency/frequency/entity/price-band)` → `Google Sheets (buyers_clean)`
   Why now: this is the "AI Cash Buyer Bot" pattern circulating this quarter — feeds directly into the existing WHL-02 Cash-Buyer Dispo Blast, making the match quality meaningfully better.
   Fit: Medium, $2,500–$3,500 (positions as a WHL-02 upgrade/precursor).

4. **Two-Way SMS Reply Classifier**
   Trigger → Node chain: `Webhook (inbound SMS/email reply)` → `Claude (classify: hot / meets-criteria / stop / wrong-number)` → `Switch (4 branches)` → `CRM tag update` → `Slack alert to acquisitions rep (hot only)` → `Suppress list update (stop)`
   Why now: the closing half of the AI Cash Buyer Bot flow above and a near-identical pattern to the Sales "Cold-Reply Classifier" — same skeleton, WHL-specific branching. High reuse value across both verticals.
   Fit: Simple–Medium, $1,800–$2,500.

5. **GHL Call-Recording Auto-Summary for Acquisitions**
   Trigger → Node chain: `Webhook (GoHighLevel call ended)` → `Download recording` → `Whisper transcribe` → `GPT summarize (seller motivation, price expectation, timeline)` → `Push note to GHL contact timeline`
   Why now: identical chain to the Sales idea above, sourced from the same NextAutomation pack — since many WHL clients already run GoHighLevel, this is close to a drop-in reuse of build work from a Sales engagement.
   Fit: Simple, $1,500–$2,200.

## Productivity (PRD)

1. **Cross-Channel Executive Priority Digest**
   Trigger → Node chain: `Schedule Trigger (8 AM weekdays)` → `Gmail (unread/priority-labeled)` → `Google Calendar (last 3 days meetings)` → `Fireflies API (transcripts for recorded meetings)` → `Code (flag meetings with no completed follow-up)` → `Claude (cross-reference to-do sheet, build ranked action plan)` → `Google Sheets append` → `Slack DM`
   Why now: a deeper version of the existing "built" 7 AM Daily Brief — [n8n's own AI Personal Assistant template](https://n8n.io/workflows/4723-ai-personal-assistant/) adds the "did this meeting actually get followed up on" check that a plain calendar/task digest misses.
   Fit: Complex, $3,500–$4,500 (premium upsell above PRD-03).

2. **Live Meeting Notetaker → Supabase**
   Trigger → Node chain: `Recall.ai bot joins call` → `Real-time transcript stream` → `OpenAI (summarize + extract action items)` → `Supabase/Postgres insert` → `Notion/Slack recap post`
   Why now: called out as one of the ["best 9 n8n AI agent examples" for 2026](https://www.jotform.com/ai/agents/n8n-ai-agent-workflow-example/) — a real-time alternative build path to PRD-02's webhook-triggered version, better for clients who want the bot to actually attend the call.
   Fit: Medium, $2,500–$3,500.

3. **Invoice/Receipt Vision Extractor**
   Trigger → Node chain: `Gmail label watch` → `Download PDF attachment` → `GPT-4o Vision (extract vendor/amount/date/line items/GST-VAT)` → `IF (confidence check)` → `Google Sheets append (high confidence)` / `Slack flag for human review (low confidence)`
   Why now: shows up independently in both [Intuz's](https://www.intuz.com/blog/best-n8n-workflow-templates/) and [dev.to's](https://dev.to/emperorakashi20/12-n8n-workflow-templates-you-can-import-and-use-today-2026-edition-2084) 2026 template lists — a tighter, finance-specific sibling to the existing PRD-05 generic PDF-to-sheet card.
   Fit: Medium, $2,600–$3,600.

4. **Proposal Follow-Up Nudge Timer**
   Trigger → Node chain: `Webhook/Gmail (proposal sent, tagged)` → `Wait 3 days` → `IF (no reply detected)` → `Slack reminder` → `Wait 4 more days` → `IF (still no reply)` → `Claude (draft follow-up email)` → `Gmail draft for review`
   Why now: from [iloveblogs' 18-template freelancer/solopreneur pack](https://www.iloveblogs.blog/guides/n8n-workflow-templates-freelancers) — a small, fast-to-build Simple-tier automation, good as a low-friction upsell add-on for existing PRD clients.
   Fit: Simple, $1,200–$1,800.

5. **Weekly Ops Report Auto-Compiler**
   Trigger → Node chain: `Schedule Trigger (Monday AM)` → `HTTP Request (Stripe/QuickBooks data)` → `HTTP Request (CRM pipeline data)` → `Code (merge + compute deltas)` → `Claude (draft formatted report copy)` → `Google Docs create from template` → `Drive save + Slack link`
   Why now: matches the "Doc & Report Generator" pattern already spec'd as PRD-04, but scoped specifically to weekly revenue/ops reporting — a concrete first build if PRD-04 gets picked up.
   Fit: Complex, $3,200–$4,200.

## Small Business (SMB)

1. **Pre-Appointment Info/Doc Verification Check**
   Trigger → Node chain: `Schedule Trigger (daily)` → `HTTP Request (upcoming appointments, next 48h)` → `IF (required doc/info on file?)` → `Notion/Airtable (create prioritized front-desk task if missing)` → `Email (client reminder to submit)`
   Why now: from [n8nlab's private-practice automation pack](https://n8nlab.io/blog/best-n8n-automations-private-practices) — generalizes past clinics to any service business with a pre-appointment requirement (contracts, deposits, ID, measurements).
   Fit: Simple–Medium, $1,600–$2,400.

2. **Aging-Bucket Billing Escalation Ladder**
   Trigger → Node chain: `Schedule Trigger (weekly)` → `HTTP Request (QuickBooks/Xero overdue invoices)` → `Code (bucket: 14/30/60 days)` → `Switch (3 branches)` → `Email (polite 14d / firm 30d)` → `Slack escalation to owner (60d)`
   Why now: a direct upgrade of the existing SMB-02 Invoice→Payment card — the flat "reminder sequence" becomes tone-aware and owner-escalating, matching what [n8nlab's aging-bucket pattern](https://n8nlab.io/blog/best-n8n-automations-private-practices) and [Scalahosting's business-owner list](https://www.scalahosting.com/blog/n8n-workflow-templates/) both converge on.
   Fit: Medium, $2,000–$2,800.

3. **Post-Service Care & Rebooking Nudge**
   Trigger → Node chain: `Webhook (appointment marked completed)` → `Wait 24h` → `Claude (personalized post-service care message)` → `Email send` → `Wait 7 more days` → `IF (rebooked?)` → `Email (gentle rebooking prompt with scheduling link)`
   Why now: same n8nlab pack — pairs naturally with the existing SMB-05 No-Show Reducer but targets clients who *did* show up, closing the retention gap on the other side.
   Fit: Simple–Medium, $1,600–$2,200.

4. **WhatsApp Instant Confirmation with Tracking/Directions**
   Trigger → Node chain: `Webhook (new order/booking)` → `Code (format details)` → `WhatsApp Business API (send confirmation + tracking link or directions)`
   Why now: cited in [dev.to's 2026 template list](https://dev.to/emperorakashi20/12-n8n-workflow-templates-you-can-import-and-use-today-2026-edition-2084) as a high-completion-rate channel — cheap, fast build, strong fit for local service businesses whose clients live in WhatsApp/SMS more than email.
   Fit: Simple, $900–$1,400.

5. **New-Client Welcome & Auto-Provisioning**
   Trigger → Node chain: `Webhook (contract signed — Docusign/PandaDoc)` → `Gmail (personalized welcome email)` → `Google Drive (create client folder + subfolders: Briefs/Deliverables/Invoices)` → `CRM (create record)` → `Task manager (create project)`
   Why now: from [iloveblogs' freelancer pack](https://www.iloveblogs.blog/guides/n8n-workflow-templates-freelancers) — distinct from the existing SMB-03 New-Hire Onboarding (that's for employees; this is for *clients*), and a natural pair with SMB-01 Client Intake.
   Fit: Medium, $1,800–$2,600.

## How this run was produced

Scraped via the Anakin scraper MCP against `n8n.io/workflows` plus a targeted
search sweep per vertical (10 results each). Cross-referenced against
`references/business-model.md`, `references/templates.md`, and
`references/pilot-plan.md` in the `n8n-automation-business` skill for pricing
and fit. Saved to this folder and mirrored into `src/data/brainstormIdeas.ts`
(kept unwired from the live catalog — see the folder `README.md`).

**Scheduling note:** this run was produced by a background agent set up to
scrape daily at ~12:00 AM. The session-level cron tool available to this agent
(`CronCreate`) is **session-only** — it does not survive past this session and
auto-expires after 7 days even if the session stays alive. For a durable daily
job, set up a **Trigger** in Claude Code on the web pointed at this repo with a
`0 0 * * *`-style schedule and this same brief; that is the persistence layer
built for recurring runs like this one. A session-scoped cron was created
alongside this report as a best-effort bridge — see the PR description for
its ID and expiry.
